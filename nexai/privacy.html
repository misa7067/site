Политика конфиденциальности

Ваше доверие — наш актив. Здесь описано, как NexAI работает с вашими данными.

1. Сбор данных

При использовании сайта и Telegram-бота мы можем обрабатывать:

  • Данные аккаунта Telegram: Username, ID (для технического обеспечения работы бота).
  • Контекст задачи: Текст, голос и изображения, которые вы отправляете боту для оценки проекта.

2. Использование Искусственного Интеллекта (AI)

Для анализа ваших задач и формирования оценки мы используем технологии LLM (Large Language Models), предоставляемые компанией OpenAI.

Важно: Отправляя данные в бот, вы соглашаетесь на их автоматизированную обработку алгоритмами ИИ. Мы настроили систему так, чтобы данные использовались только для генерации ответа и не сохранялись для дообучения моделей (политика Zero Data Retention, где это возможно).

3. Зачем нам это нужно

Исключительно для того, чтобы:

  • Сформировать для вас техническое задание.
  • Рассчитать бюджет проекта.
  • Связаться с вами для начала работы.

4. Безопасность

Мы не продаем ваши контакты спам-базам. Доступ к вашим проектам имеют только инженеры NexAI под NDA (соглашением о неразглашении).

5. Контакты

Вы можете запросить удаление всей истории переписки в любой момент.
