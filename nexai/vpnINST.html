Ручная настройка

В случае, если по какой-то причине не сработал автоматический импорт, выполните настройку ниже.

01 // INIT

Вход в меню

Откройте бот NexVPN в Telegram.
Введите команду /start для перезагрузки интерфейса и вызова главного меню.

02 // GET KEY

Копирование конфигурации

1. Нажмите кнопку «Установить VPN».
2. В появившемся сообщении вы увидите ключ подключения (набор символов).
3. Просто нажмите на код ключа — он автоматически скопируется в буфер обмена вашего устройства.

03 // MANUAL IMPORT

Импорт в приложение

Вернитесь в приложение v2RayTun.
Нажмите кнопку «+» (обычно в углу экрана), затем выберите пункт «Добавить из буфера» (Import from clipboard).

04 // CONNECT

Активация

Для Включения/Выключения VPN нажимайте большую круглую кнопку, находящуюся в центре экрана (как показано на изображении выше).
